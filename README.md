# ProductManagement
A RESTful API for managing products, including functionalities for creating, updating, retrieving, and deleting products.

# Requirements
Software:
JDK 17 or later
Maven 3.8+ or Gradle
IDE (e.g., IntelliJ IDEA, Eclipse)
Git (optional)

# Running the Application
Default Credentials
